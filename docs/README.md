# POO Project:

## Descripción
Subproyecto desarrollado para explicar conceptos de Programación Orientada a Objetos en Java mediante una página web interactiva.

## Objetivo
Facilitar el aprendizaje de conceptos fundamentales de POO a través de ejemplos, ejercicios y una interfaz visual amigable, donde el usuario/estudiante podrá acceder a los recursos con facilidad.

## Herramientas con las cuales se creó el proyecto:
- HTML5
- CSS3
- JavaScript
- Tailwind CSS
- Lucide Icons
